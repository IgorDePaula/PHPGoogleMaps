#PHP Wrapper for Google Maps API v3

[Examples](http://www.galengrover.com/projects/php-google-maps/examples/)

PHP 5.3+

##Features

 - Event Listeners
 - DOM Event Listeners
 - Fusion Tables
 - KML Layers
 - Directions
 - Custom Map Styles
 - Markers
 - Custom Marker Icons
 - Marker Groups
 - Shapes
 - Geocoding
